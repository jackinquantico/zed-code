테두리 관련 스타일 (border~)

요소의 테두리 영역 (border) 의 스타일을 지정할 때 쓰임
테두리의 두께, 색상, 선의 모양, 각진 정도, .. 

            [ 표현법 ]
            선택자 {

                * 테두리 스타일
                border-위치-style : none / dotted / dashed / solid / double / groove / ridge / inset / outset;

                * 테두리 두께
                border-위치-width : 테두리두께(px);

                * 테두리 색상
                border-color : 색상;

                * 테두리 두께, 스타일, 색상
                border-위치 : 두께 스타일 색상; => 색상 생략 가능(생략시 글자색으로 통일)

                * 테두리 모서리의 각진 정도
                border-위아래-좌우-radius : 숫자값(px); => 숫자가 클 수록 곡선에 가까워짐

                * 박스에 그림자 효과
                box-shadow : 가로거리(x) 세로거리(y) 흐림정도 번짐정도 그림자색상;
            }

            => 위치는 위/아래/좌/우(top/bottom/left/right) 지정 가능, 생략 가능
